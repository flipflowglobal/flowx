# Ultimate Aureon-v Orchestrator

Fully operational orchestrator with:
- gRPC server for health, module list, start/stop
- Prometheus metrics on port 8000
- Vault and NATS integration
- Docker Compose and Dockerfile
- GitHub Actions CI pipeline
- Real module implementations (WalletCore, PredictFlip, RevenueHive, ComplianceGuard, WatchDog)

## Quickstart

```bash
docker compose up -d --build
pytest tests
grpcurl -plaintext localhost:50051 orchestrator.OrchestratorService/Health
```
