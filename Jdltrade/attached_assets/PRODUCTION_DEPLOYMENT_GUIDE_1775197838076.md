# AUREON — Production Deployment Guide

This guide covers every deployment path: bare-metal, Docker, and cloud VM.

---

## Quick Start (bare-metal)

```bash
git clone https://github.com/flipflowglobal/D.L.git
cd D.L
bash deploy.sh          # installs deps, creates venv, guides wallet setup
python trade.py         # paper trading (no funds at risk)
python trade.py --live  # live mainnet trading
```

| Tool | Minimum | Install |

| git | any | `sudo apt install git` |
| RPC endpoint | — | Free at [alchemy.com](https://alchemy.com) or [infura.io](https://infura.io) |

## Bare-Metal Setup

### 1. Clone and install

```bash
git clone https://github.com/flipflowglobal/D.L.git
cd D.L

python3 -m venv venv
source venv/bin/activate

pip install -r requirements.txt
```

### 2. Wallet setup (run once)

```bash
python setup_wallet.py
```

Generates or imports a wallet, saves to `vault/wallet.json` (chmod 600), and patches `.env`.

### 3. Configure `.env`

```bash
cp .env.example .env
nano .env   # set RPC_URL to your Alchemy/Infura HTTPS endpoint
```

### 4. Run

```bash
# Paper trading (safe — no transactions)
python trade.py

# Live mainnet trading
python trade.py --live

# FastAPI cognitive agent server
uvicorn main:app --host 0.0.0.0 --port 8010

# DL_SYSTEM quest automation
python DL_SYSTEM/main.py
```

### 5. Run as background daemon

```bash
nohup python trade.py > logs/trade.log 2>&1 &
echo $! > trade.pid

# Stop
kill $(cat trade.pid)
```

## Docker Deployment

### Build and start all services

```bash
cp .env.example .env
nano .env   # add RPC_URL and wallet credentials

docker compose up -d
```

Services started:
- `aureon-api` — FastAPI server on port 8010
- `aureon-bot` — Trading bot in paper mode

### Useful commands

```bash
docker compose logs -f aureon-bot   # live log stream
docker compose stop                  # stop all
docker compose down                  # stop and remove containers
docker compose ps                    # check status
```

### Live trading in Docker

```bash
TRADE_MODE=live docker compose up -d aureon-bot
```

Or set `TRADE_MODE=live` in your `.env`.

## Cloud VM (Ubuntu 22.04+)

```bash
# 1. Provision VM (2 vCPU / 2 GB RAM minimum)
# 2. SSH in and clone
sudo apt update && sudo apt install -y python3 python3-pip python3-venv git
git clone https://github.com/flipflowglobal/D.L.git
cd D.L

# 3. Deploy
bash deploy.sh

# 4. Keep running after SSH disconnect
nohup uvicorn main:app --host 0.0.0.0 --port 8010 > logs/api.log 2>&1 &
nohup python trade.py > logs/trade.log 2>&1 &
```

## Environment Variables Reference

See `.env.example` for full documentation of every variable.

**Required for live trading:**
- `RPC_URL` — Ethereum mainnet RPC (Alchemy/Infura HTTPS)
- `PRIVATE_KEY` — 64-char hex private key (set by `setup_wallet.py`)
- `WALLET_ADDRESS` — Ethereum address (set by `setup_wallet.py`)

**Optional tuning:**
- `TRADE_SIZE_ETH=0.05` — ETH per trade
- `SCAN_INTERVAL=30` — seconds between cycles
- `MAX_DAILY_TRADES=20` — risk guard

## Security Checklist

- [ ] `vault/wallet.json` has `chmod 600` — check with `ls -la vault/`
- [ ] `.env` is NOT committed — verify `git check-ignore -v .env`
- [ ] Private key never in git history — check `git log --all -p | grep PRIVATE_KEY`
- [ ] Paper mode tested for ≥ 1 hour before going live
- [ ] `GAS_BUDGET_USD` set to prevent runaway gas costs
- [ ] `MAX_DAILY_TRADES` capped to limit exposure
